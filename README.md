# **JET- SIMPLEX - SEARCH**
`jet_simplex_search` is a Python package that uses that [`state_collapser`](https://github.com/TYLERSFOSTER/state_collapser) package to give logarithmic speed-up to the search for $k$-simplices in a given graph.
